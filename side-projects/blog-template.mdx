> **"지루한 글쓰기 노동은 AI에게, 당신은 경험에만 집중하세요."**
> 
> 사용자의 과거 게시글을 심층 분석하여 고유의 '문체(Persona)'를 재현하고,  
> 간략하게 작성한 문장들을 1,500자 이상의 완성된 포스팅으로 확장해주는 **AI 고스트라이터 서비스** 입니다.

## 🚀 프로젝트 배경
단순한 "키워드 기반 자동 생성"이 아닙니다.  
많은 블로거들이 겪는 **"경험은 있는데 글로 풀어내기 귀찮은"** 문제에 집중했습니다.  
내 말투를 그대로 흉내 내는 AI가 초안을 이어받아 작성함으로써, **생산성을 10배 이상** 높이는 것을 목표로 합니다.

## ✨ 핵심 기능 구현

**1. 초개인화 스타일 엔진**
- **Style DNA 분석**: Cheerio로 기존 블로그 글을 크롤링하고, Claude Sonnet 4.5로 문체, 감정선, 문단 호흡 등을 정밀 분석하여 JSON 프로필로 추출
- **Few-shot Prompting**: 추출된 스타일 데이터를 기반으로 블로거 고유의 말투를 완벽하게 모방

**2. 사실 기반 증강 작문**
- **Draft Expansion**: 사용자가 입력한 짧은 경험(메모)을 바탕으로 내용을 확장하여, Hallucination 없는 진정성 있는 글 작성
- **Real-time RAG**: Tavily API를 통해 가게 위치, 영업시간, 메뉴 가격 등 최신 정보를 실시간으로 검색해 내용 보강

**3. 몰입형 UX**
- "누구와 갔나요?", "어떤 메뉴를 드셨나요?" 등 대화하듯 자연스러운 단계별 입력
- Next.js Server Actions와 `useFormStatus`를 활용한 부드러운 인터랙션

## ⚙️ 기술 구현

![](/img/sideProject/blog-template/i54pmg3.png)

### 🏗️ **아키텍처 (Architecture)**
- **Framework**: Next.js 16 (App Router) 기반의 최신 React 아키텍처
- **State Management**: Zustand 및 React Query 없이, Server Components와 Native Fetch만으로 간결한 상태 관리 구현
- **Database**: Supabase (PostgreSQL)를 도입하여 사용자 데이터 영속성 및 확장성 확보 (Local-First에서 마이그레이션)
- **Design Pattern**: FSD (Feature-Sliced Design)를 적용하여 유지보수성과 모듈 간 결합도 최적화

### 🤖 **AI Pipeline**  
```
RSS Crawling → Style Analysis → RAG Context → Text Generation
```
- Analysis: Claude Sonnet 4.5 (스타일 분석)
- Generation: Claude Haiku 3.0 (리뷰 생성)
- Search: Tavily AI Search API (실시간 정보)

### 🛠️ Tech Stack
- **Frontend** : Next.js 16, TypeScript, Tailwind CSS 4
- **Backend/DB**: Supabase (Auth, Database), NextAuth.js
- **AI/Data**: Anthropic Claude API, Tavily Search API, Cheerio
