> 사용자의 과거 게시글을 학습해 고유의 문체를 모방하고,  
실시간 정보 검색(RAG)을 통해 사실 기반의 글쓰기를 돕는 Next.js 16 기반의 웹 애플리케이션입니다.


## 🚀 프로젝트 배경
체험단 활동이나 정기 블로그 포스팅이 부담스러운 사용자를 위한 AI 에디터입니다.  
- **핵심 가치**: "내 말투로, 거짓말 없이, 누구나 쉽게"


## ✨ 핵심 기능 구현

**1. 스타일 학습 (Hyper-Personalization)**
- Cheerio로 네이버 블로그 RSS 피드 크롤링 및 HTML 파싱
- Claude Sonnet 4.5로 사용자의 어조, 이모지 사용 패턴, 문단 구조 분석
- Few-shot Learning: 작성한 이전 글 들을 참조하여 문체 재현

**2. 사실 기반 글쓰기 (RAG)**
- Tavily Search API로 메뉴, 영업시간, 주차 정보 등 실시간 수집
- 검증된 정보만 활용하여 AI Hallucination 방지
- 1,500자 이상의 사실 기반 리뷰 생성

**3. 대화형 UI (Flow-Based UX)**
- Wizard 패턴으로 단계별 정보 입력 유도
- 자연어 명령("좀 더 신나게 바꿔줘")으로 특정 문단 수정
- 실시간 프리뷰 및 편집 기능

**4. 로컬 우선 프라이버시 (Local-First)**
- 모든 분석 데이터 및 생성된 리뷰는 로컬 파일 시스템에만 저장
- 외부 서버로의 개인 데이터 전송 없음

## ⚙️ 기술 구현

![](/img/sideProject/blog-template/i54pmg3.png)


### 🏗️ **아키텍처 설계**
- FSD (Feature-Sliced Design) 패턴으로 비즈니스 로직(Features)과 UI(Widgets) 레이어 분리
- App Router 구조로 페이지 라우팅과 기능 모듈 분리

### 🤖 **AI Pipeline**  
```
RSS Crawling → Style Analysis → RAG Context → Text Generation
```
- Analysis: Claude Sonnet 4.5 (스타일 분석)
- Generation: Claude Haiku 3.0 (리뷰 생성)
- Search: Tavily AI Search API (실시간 정보)

### 🛠️ **Tech Stack**
- Next.js 16 App Router, React 19, TypeScript
- Tailwind CSS 4로 반응형 UI 구현
- Native Fetch API 기반 비동기 상태 관리
- Claude API 사용
