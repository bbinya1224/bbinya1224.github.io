### Frontend Engineering Experience
- Next.js 16 & React 19 활용:
  - App Router 아키텍처: 페이지(app)와 기능(features)을 분리하여 응집도 높은 디렉토리 구조를 설계했습니다.
  - Tailwind CSS 4: 차세대 CSS 엔진을 적용하여 런타임 오버헤드 없는 고성능 스타일링 시스템을 구축했습니다.
- 지속 가능한 아키텍처 설계 (FSD): 복잡한 AI 로직을 효율적으로 관리하기 위해 FSD(Feature-Sliced Design) 패턴을 도입, 비즈니스 로직(Entity)과 UI(Widget) 간의 의존성을 30% 이상 낮췄습니다.
- AI Native UI/UX 구현:
  - Interactive Review Wizard: 사용자가 AI와 대화하듯 단계별로 입력을 완료하는 Wizard UI를 직접 구현하여 복잡한 입력 과정을 단순화했습니다.
  - Optimized Data Fetching: 무거운 라이브러리 없이 Native Fetch API와 React Hooks(useEffect) 만으로 비동기 상태 관리 로직을 최적화하여 번들 사이즈를 최소화했습니다.

### Product Solving

- RAG 기반 신뢰성 확보:  
  Tavily Search API를 통해 가게 정보를 실시간 검증, AI의 환각(Hallucination) 현상을 방지하고 콘텐츠 신뢰도를 높였습니다.
- Personalization Engine:  
  Cheerio로 기존 블로그를 분석하여 이모지와 말투를 학습, 정형화된 AI 톤을 벗어난 "나만의 글쓰기" 경험을 제공했습니다.

### Tech Stack 
Next.js 16, TypeScript, React 19, Tailwind CSS 4, FSD Architecture, Claude API
